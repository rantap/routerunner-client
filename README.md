# React routeplanner/workout log

## [Open project](https://routerunner.netlify.app/)

Plan your running/walking routes and add them to your workout log (under construction)

## Tech stack

- TypeScript
- React
- React Aria Components
- Tailwind CSS
- MapLibre

## Installation

`npm i`

## Run

`npm run dev`
